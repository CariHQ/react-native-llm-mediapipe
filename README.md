# react-native-llm-mediapipe

Run an LLM on iOS & Android devices using React Native

## Installation

```sh
npm install react-native-llm-mediapipe
```

## Usage

```js
import { multiply } from 'react-native-llm-mediapipe';

// ...

const result = await multiply(3, 7);
```
